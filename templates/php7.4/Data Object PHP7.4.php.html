This is a template used to create new PHP Data Object classes using PHP version 7.4.
Predefined variables will take the following values:
${NAME}, ${CLASSNAME}, ${CLASS_NAME}   Name of the PHP data class.
${NAMESPACE}   Namespace of the PHP data class.
${IMPORTS}   Array of Fully Qualified class names used within the class for use in import/use statements.
${BUILDER_CLASSNAME}   The class name of the Builder instance.
${IS_IMMUTABLE}   True if setter methods are to be generated.
${CLASS_PROPERTIES}   Array of PHP class properties defined on the class.
Example Usage: 
    #foreach( ${PROPERTY} in ${CLASS_PROPERTIES} )
        // get the name of the property.
        ${PROPERTY.getName()}
    #end
The following methods are defined on each ${CLASS_PROPERTIES} instance:

Accessors

  • [String] getScope() : The scope of the property ("private", "protected", or "public")
  • [Boolean] isNullable() : True if the property is defined as nullable.
  • [String] getFQType() : Fully Qualified class type of the property, including namespace.
  • [String] getType() : Class type of the property, does NOT include the namespace of the class.
  • [String] getNamespace() : Namespace of the class the property is defined as.
  • [String] getName() : Name of the property.
  • [String] getDefaultValue() : The default value the property is defined as having.

State Checking

  • [Boolean] isNullable() : True if the property allows nulls, false if not.
  • [Boolean] hasType() : True if the property has a type defined. False if no type is defined.
  • [Boolean] hasNamespace() : True if the property's classtype has an associated namespace.
  • [Boolean] hasDefaultValue() : True if the property has a default value associated.

Code Generator Helpers

  • [String] asMethodParameter() : Definition of the property as used as a method parameter, including any default value if defined.
  • [String] asMethodParameterWithoutDefaultValue() : Definition of the property as used as a method parameter, not including a default value if defined.
  • [String] asClassPropertyDeclaration() : Definition of the property as used as in a class definition statement.
  • [String] asPhpDocDeclaration() : Definition of the property as used in a PhpDoc statement.
  • [String] getNameCapitalized() : Name of the property, in UpperCaseSnakeFormat.
The following standard template variables can also be used:
${FILE_NAME}   Current file name.
${USER}   Current user system login name.
${DATE}   Current system date.
${TIME}   Current system time.
${YEAR}   Current year.
${MONTH}   Current month.
${DAY}   Current day of the month.
${HOUR}   Current day of the month.
${DAY}   Current hour.
${MINUTE}   Current minute.
${PRODUCT_NAME}   Current IDE name.
${PROJECT_NAME}   Current project name
${DS}   Dollar sign, evaluates to a plain '$' character.
